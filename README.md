# Smart Duken

## Description
**Smart Duken** is a software solution designed to automate supply planning, financial accounting, supplier and customer management, as well as business analytics for small and medium-sized enterprises.

The software is implemented as a web-based platform and follows the Software as a Service (SaaS) model, providing users with centralized access to functionality via a web interface.

---

## Purpose
The primary purpose of the software is to digitalize and automate business processes related to supply management, cash flow tracking (incoming and outgoing funds), counterparty management, and financial performance analysis.

The project was initially developed for use in retail operations; however, its architecture and functionality allow it to be applied to other business domains with similar operational processes.

---

## Core Functionality
- supply planning and management;
- supplier and customer accounting;
- financial transaction tracking (cash in / cash out);
- income and expense analysis;
- generation of analytical data and reports;
- user management and access control;
- automation of internal business processes;
- asynchronous background task processing;
- structured data storage and processing.

---

## Architecture and Technologies
The software is implemented as a web application with a client-server architecture.

### Backend:
- Python
- Django
- Django REST Framework
- Celery
- Redis
- PostgreSQL

### Frontend:
- TypeScript
